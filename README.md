# HillCity Advertising - React Web Application

## Project Overview

**HillCity Advertising** is a modern, interactive web application for a leading outdoor advertising agency in Kandy. Built with **React, Tailwind CSS, and Framer Motion**, the platform showcases the agency’s services, brand activations, sustainability projects, and contact information in a professional, visually appealing layout.

This project demonstrates **state-of-the-art frontend development practices** with a focus on animation, responsiveness, and user experience.

---

## Features

- **Responsive Design** – Works seamlessly on mobile, tablet, and desktop devices.
- **Animated UI** – Smooth animations powered by **Framer Motion** for engaging user interactions.
- **Service Showcases** – Detailed service sections with images, banners, and cards highlighting agency capabilities.
- **Events & Promotions** – Display sports, corporate, and community events with structured layouts.
- **Contact Us Section** – Fully designed contact page with Google Maps integration, email, phone, and working hours.
- **Social Media Integration** – Facebook, Twitter/X, Instagram, LinkedIn icons linked to agency profiles.
- **Navbar & Footer Components** – Fully functional navigation with mobile dropdowns and top bar info.

---

## Tech Stack

- **Frontend:** React, JSX
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment Ready:** Can be hosted on GitHub Pages, Netlify, or Vercel
